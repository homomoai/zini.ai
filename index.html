How to chat with Zini AI – Step by Step

🎥 Watch on YouTube App ↗
  1. Open any browser on your phone (like Chrome).
  2. In the search bar, type: groq cloud
  3. Click on the first result (the official Groq Cloud website).
  4. Tap on "Sign in" or "Get Started". Log in with your Google account.
  5. In the top right corner, tap the button with three icons.
  6. Go to API Keys. Tap Create new API key. Give it a name: zini ai. Then create it.
  7. Copy the long API key that appears.
  8. Open the Zini AI app. Paste the API key there.
